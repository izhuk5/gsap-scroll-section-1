# Happy Scroll

![Scroll Animation](./scroll-gif.gif)

Interactive page with animated scroll.

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- GSAP for animations
- Inter font

## Features

- Responsive design
- Fluid typography
- Scroll animations
- Snap scroll

## Installation

1. Clone the repository
2. Open `index.html` in your browser

## Usage

Just scroll down to view animations.
